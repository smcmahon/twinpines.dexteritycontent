twinpines.dexteritycontent Installation
---------------------------------------

To install twinpines.dexteritycontent using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``twinpines.dexteritycontent`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        twinpines.dexteritycontent
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
